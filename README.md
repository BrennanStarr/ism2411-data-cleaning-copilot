# ism2411-data-cleaning-copilot

This small project demonstrates a simple data cleaning pipeline for a messy sales CSV.

Run the cleaning script from the project root:

```bash
python src/data_cleaning.py
```

The script reads `data/raw/sales_data_raw.csv` and writes `data/processed/sales_data_clean.csv`.

Files added:
- `src/data_cleaning.py` — main cleaning script with Copilot-assisted functions
- `data/raw/sales_data_raw.csv` — sample messy data (for demo/testing)
- `data/processed/sales_data_clean.csv` — generated by running the script
- `reflection.md` — short writeup about using Copilot
